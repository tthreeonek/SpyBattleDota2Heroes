# 🕵️ Telegram bot для игры ШПИОН с героями Dota 2.
___
## 📦 Требования
- Python 3.8 или выше

- pip (менеджер пакетов)

- Аккаунт в Telegram и токен бота
___
## 🛠 Установка
1. Клонируйте репозиторий
```bash
git clone https://github.com/tthreeonek/SpyBattleDota2Heroes.git
cd SpyBattleDota2Heroes
```
___
## Запуск
___
### 1. Установите зависимости:
```bash
   pip install -r requirements.txt
```
### 2. Создайте файл ```.env``` в папке SpyBattleDota2Heroes и вставьте туда токен бота и юзернейм админа в файле:
```text
BOT_TOKEN=token
ADMIN_USERNAME=username

# Настройки прокси (пример для SOCKS5)
PROXY_TYPE=socks5
PROXY_HOST=ваш_прокси_хост  # например 127.0.0.1 или адрес прокси
PROXY_PORT=1080
PROXY_USERNAME=  # если нужна авторизация, иначе оставьте пустым
PROXY_PASSWORD=  # если нужна авторизация
```
### 3. Запустите бота:
```bash
python main.py
```
___
## Как играть

1. Каждый игрок пишет боту в личку ```/start```
2. Добавь бота в групповой чат Telegram
3. В группе используй ```/start_game```
4. Для остановки игры: ```/end_game```
___
## Команды

- ```/start``` - регистрация игрока (только в личке)
- ```/start_game``` - старт игры (только в группе)
- ```/vote``` - голосование в группе (```/vote``` @username или ответом на сообщение)
- ```/end_game``` - завершить игру (только в группе)
- ```/admin_games``` - Показать все активные игры (только для админа)
___
## Данные героев и картинки

- Список героев: `data/heroes.json`
- Картинки: `assets/heroes/*.png`
___
## 🔌 Настройка прокси (если Telegram блокируется)
Если вы находитесь в регионе с ограничениями, то включите VPN или добавьте прокси в файл ```.env:```
```text
PROXY_TYPE=socks5      # или http
PROXY_HOST=127.0.0.1
PROXY_PORT=1080
PROXY_USERNAME=        # если нужна авторизация, иначе оставьте пустым
PROXY_PASSWORD=
```
И установите дополнительную зависимость для SOCKS5:
```bash
pip install "httpx[socks]"
```
После этого бот будет использовать прокси для всех запросов к Telegram API.

## 📁 Структура проекта
```
SpyBattleDota2Heroes/
├── main.py               # Основной код бота
├── config.py             # Чтение переменных окружения и путей
├── requirements.txt      # Зависимости
├── .env                  # Секреты (токен, прокси) – не попадает в Git
├── data/
│   ├── heroes.json       # Список героев (обязательно)
│   └── registrations.json # Автоматически создаётся (список игроков)
└── assets/               # Необязательная папка для картинок героев
    └── heroes/           # .png файлы с именами из heroes.json
```
